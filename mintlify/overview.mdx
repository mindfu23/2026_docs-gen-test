## What this is

**Open5e** is an open REST API for 5th-edition d20 game content — the SRD plus other OGL/CC
material: spells, creatures, classes, magic items, and more. It is **completely open: no API
key, no auth.** The current surface is **v2**; responses are JSON, paginated, and filterable.

## Base URL

```
https://api.open5e.com
```

## Your first request

<CodeGroup>
```bash cURL
curl "https://api.open5e.com/v2/spells/?name=Fireball"
```

```python Python
import requests
r = requests.get("https://api.open5e.com/v2/spells/", params={"name": "Fireball"})
spell = r.json()["results"][0]
print(spell["name"], "— level", spell["level"])
```
</CodeGroup>
